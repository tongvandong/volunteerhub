# Pages

Moi file trong day la mot man hinh lon hoac mot tab/man hinh con.

```text
admin/       # Man hinh quan tri
auth/        # Login, register, forgot/reset password
organizer/   # Man hinh cho nha to chuc
public/      # Landing, event list/detail, verify certificate
shared/      # Man hinh dung chung sau dang nhap
sponsor/     # Man hinh nha tai tro
volunteer/   # Man hinh tinh nguyen vien
```

Muon tim mot page tu URL, xem `src/app/routes/routeConfig.jsx` truoc.
