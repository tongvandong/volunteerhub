# Components

```text
layouts/  # Khung trang: MainLayout, PublicLayout, SharedLayout
ui/       # Component nho dung lai nhieu page
```

Quy tac debug:

1. Neu loi chi xay ra trong mot man hinh, xem `pages/` truoc.
2. Neu loi lap lai o nhieu man hinh, xem component trong `components/ui/`.
3. Neu loi menu/header/sidebar/public shell, xem `components/layouts/`.
